# excel-projesi
Excel ile veri analizi, raporlama ve formül uygulamalarımı gösteren projeler Excel projelerim: fonksiyonlar, pivot table, dashboard ve veri işleme örnekleri

# Excel Projeleri

Bu repository, Excel ile yaptığım veri analizi ve raporlama projelerini içerir:
- Formüller (ÇOKEĞERMAX, ÇOKEĞERMİN, DÜŞEYARA, EĞER, VE/VEYA/YADA)
- Dashboardlar
- Pivot Tablolar
- Power Query
- Koşullu Biçimlendirme ve Veri Doğrulama
- Veri işleme teknikleri (Metni Sütunlara Dönüştür, KAYDIR, PARÇAAL)
